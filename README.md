## Install
Install from xxh repo:
```
xxh +I xxh-plugin-fish-ohmyfish
```
Install from any repo:
```
XXH +I xxh-plugin-fish-ohmyfish+git+https://github.com/xxh/xxh-plugin-fish-ohmyfish
```
Connect:
``` 
xxh yourhost +s fish +if
```
